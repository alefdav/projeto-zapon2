PERGUNTAS FREQUENTES (F.A.Q.)

Tenho que instalar algum programa?

Sim, para a sua segurança e privacidade e para que o sistema seja integrado ao seu WhatsApp, é necessário instalar um programa no seu computador/laptop com o sistema operativo Windows.

Eu posso usar o WhatsApp que já tenho?

Sim, no entanto, recomendamos que tenha um número de WhatsApp apenas para utilizar com o sistema Zapon, e outro número de WhatsApp para atender o seu cliente caso ele queira falar diretamente com um profissional para tirar alguma dúvida.

Como eu consigo acompanhar as minhas marcações?

Você terá acesso a um painel de controlo onde é possível acompanhar em tempo real as marcações que os seus clientes fazem pelo WhatsApp. Além disso poderá visualizar a sua agenda através do comando "agenda" dentro do próprio WhatsApp.

Preciso de um computador para utilizar a Zapon?

Sim, você precisa de um computador para instalar o nosso software Zapon, o sistema precisa estar ligado ao seu WhatsApp via software. Este computador servirá como servidor de marcações 24 horas por dia / 7 dias por semana.

O que significa um "profissional" na Zapon?

Um profissional é uma pessoa que é inserida no sistema Zapon como um prestador de serviços. O seu estabelecimento poderá ter vários profissionais a receber marcações através de um único sistema centralizado.
